Application Examples

This page lists all application examples and their purposes.

Applications exist in source subdirectly, subfolder is named after application name. Note that not all applications exist for every processor architecture.

Application nameDescriptionNotes
basic interruptHow to use interrupt serviceOnly Nordic nRF52xx
ble_appHow to receive BLE beaconsOnly 2.4GHz devices
ble_beaconHow to transmit BLE beaconsOnly 2.4GHz devices
blinkVery simple blink/hello world application
bme280_exampleHow to use Bosch Sensortec BME280 environmental sensor Only Ruuvitag
custom_appSimple data transmission and reception
dualmcu_appImplementation of dual-MCU API interface, used most commonly in sinks
minimal_appMinimal app that just starts the stack
nfcNFC peripheral usageOnly Nordic nRF52xx
positioning_appApplication to acquire network data for the positioning use caseOnly Nordic nRF52xx
pwm_driverCreate PWM signalOnly Nordic nRF52xx
ruuvi_evkSend sensor data to backend, control how to send, used in evaluation kitsOnly Ruuvitag
scheduler_exampleHow to use Application scheduler
